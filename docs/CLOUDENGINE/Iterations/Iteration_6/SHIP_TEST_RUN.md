# CLOUDENGINE Ship Physics — Test Run

## Запуск

Открой cmd и выполни:
```
cd c:\CLOUDPROJECT\CLOUDENGINE\build_test\Debug
CloudEngine.exe
```

## Тестируй управление

1. **Правый клик мышью** — захват курсора
2. **W** — движение вперёд
3. **A/D** — поворот влево/вправо
4. **Q/E** — вверх/вниз
5. **Shift** — буст
6. **ESC** — выход

## Проверь глазами

- Корабль двигается вперёд по нажатию W?
- Есть ли gravity (корабль падает когда не двигаешься)?
- A/D поворачивают корабль?

## После выхода

Сообщи результат:
- "Работает" — движение есть
- "Краш" — на каком моменте
- "Корабль не двигается"
- "Другое" — опиши
