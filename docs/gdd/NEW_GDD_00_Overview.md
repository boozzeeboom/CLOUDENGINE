## Концепт игры

**Облачный Мир** — это исследовательская MMO над бескрайними облаками. Ты — пилот баржи, летающей между парящими горами.

Мир процедурный, огромный (Minecraft-стиль), с красивыми генеративными облаками в стиле аниме студии Гибли.

### Что делает игрок

1. **Летит** — на барже с антигравитацией
2. **Исследует** — огромный мир, горы, платформы
3. **Собирает** — ресурсы, артефакты
4. **Торгует** — между поселениями на пиках
5. **Выживает** — в опасных зонах

## Ключевые принципы

| # | Принцип | Почему важен |
|---|--------|--------------|
| 1 | **Простота** | Сложное = плохое. Делаем минимум, но качественно. |
| 2 | **Масштаб** | Мир огромный, как Minecraft. Исследование — основной геймплей. |
| 3 | **Облака** | Многослойные, красивые, генеративные. Это визитная карточка. |
| 4 | **Ветер** | Физика ветра важнее всего остального. |
| 5 | **Визуал** | Low-poly или voxel — неважно. Важно, чтобы работало. |

## Мир

### Структура

```
Мир круговой, радиус ~350,000 единиц
Или бесконечный (seed-based)

Высота:
- 0-1000м:    Завеса (опасность, death zone)
- 1000-2000м: Нижние облака
- 2000-4000м: Основной слой (где летаем)
- 4000-6000м: Верхние облака
- 6000м+:     Космос (редко)
```

### Что в мире

| Элемент | Описание |
|---------|----------|
| **Горы/Пики** | Процедурные, разной высоты |
| **Города** | 3-4 поселения на больших пиках |
| **Фермы** | Платформы с посевами |
| **Заброшенки** | Остовы, лут, секреты |
| **Облака** | 3 слоя, генеративные, двигаются |
| **Ветер** | Глобальный + локальные зоны |

### Завеса (Veil)

```
Завеса = ядовитый слой облаков внизу
Не пролетать — урон, смерть
Визуал: тёмно-фиолетовый, молнии
```

## Облака (Ключевая фича)

### Система

```
3 слоя:
- Upper:   4000-6000м (далёкие, low detail)
- Middle:  2000-4000м (основные, medium)
- Lower:   500-2000м  (близкие, high detail)

Рендеринг: Volumetric raymarching
Стиль: Ghibli (мягкие, тёплые, rim lighting)

Анимация: Wind-driven (ветер двигает облака)
Процедурность: Seed-based (детерминированные)
```

### Визуал

```
Base color:    Тёплый белый (#FFF8F0)
Shadow:        Прохладный синий (#CCD6E0)
Rim light:     Тёплый оранжевый (#FFB066)
Ambient:       Небесный голубой (#99B3E6)
```

## Ветровая система (Ключевая фича #2)

### Глобальный ветер

```
Всегда присутствует
Направление: меняется медленно
Скорость: 5-20 м/с (настраивается)
Влияние: сдвиг курса, турбулентность
```

### Локальные зоны ветра

| Тип | Описание | Влияние |
|-----|---------|---------|
| **Thermal** | Восходящие потоки у гор | Подъём |
| **Shear** | Резкая смена направления | Турбулентность |
| **Gust** | Порывы | Внезапные сдвиги |
| **Trade** | Постоянные маршруты | Стабильный |

## Геймплей

### Core Loop

```
1. Берём контракт/миссию
2. Загружаем баржу
3. Летим к цели (ветер!)
4. Исследуем / Доставляем / Собираем
5. Возвращаемся
6. Продаём / Тратим
7. Повторяем
```

### Кооператив

```
2-4 игрока на одной барже
Разделение ролей:
- Пилот (управление)
- Штурман (навигация)
- Грузчик (погрузка/разгрузка)
- Страж (наблюдение за опасностью)

Input averaging на сервере
```

## Scope (Что делаем в MVP)

### Делаем

```
□ Генеративный мир (chunks, seed)
□ 3 слоя облаков (volumetric)
□ Физика баржи (ветер, инерция)
□ Глобальный + локальный ветер
□ Загрузка/выгрузка чанков
□ Кооп на одной барже (2-4 игрока)
□ Базовая торговля
□ 3-4 города на пиках
```

### НЕ делаем (пока)

```
✗ Сложная экономика
✗ Полная RPG система
✗ Много контента (biomes)
✗ AI NPC с поведением
✗ Полная карта мира
✗ Мобильная версия
```
